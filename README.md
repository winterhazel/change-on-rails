## Getting Started
### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/winterhazel/change-on-rails.git
   ```
2. Install dependencies
   ```sh
   bundle
   ```
3. Initialize the database
   ```sh
   rails db:create db:migrate
   ```
4. Serve
   ```sh
   rails s
   ```
